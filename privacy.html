🔒 Resumo simples: O PageTurn não coleta, armazena nem transmite nenhum dado pessoal. Tudo fica no seu celular.

1 Informações que coletamos

O PageTurn não coleta nenhuma informação pessoal dos usuários. O aplicativo não requer criação de conta, não solicita e-mail, nome, localização ou qualquer outro dado identificável.

O aplicativo funciona completamente de forma local, sem qualquer comunicação com servidores externos.

2 Armazenamento de dados

Todos os dados gerados pelo uso do aplicativo — incluindo sua biblioteca de livros, progresso de leitura, configurações e preferências — são armazenados exclusivamente no armazenamento local do seu dispositivo Android.

Esses dados nunca saem do seu celular e não são enviados para nenhum servidor, nuvem ou serviço externo.

3 Arquivos do usuário

Os arquivos importados pelo usuário (PDFs, CBRs, CBZs e outros formatos suportados) são copiados para o armazenamento interno do aplicativo para facilitar o acesso.

Esses arquivos nunca são enviados para a internet, não são acessados por terceiros e permanecem sob controle exclusivo do usuário. Ao desinstalar o aplicativo, todos os dados são removidos do dispositivo.

4 Permissões utilizadas

O PageTurn solicita as seguintes permissões do Android:

Leitura de armazenamento — utilizada exclusivamente para permitir que o usuário selecione e importe seus próprios arquivos de leitura (PDF, CBR, CBZ). Esta permissão não é usada para acessar outros arquivos do dispositivo.

5 Serviços de terceiros

O PageTurn não utiliza serviços de análise, publicidade, rastreamento, relatórios de erro ou qualquer outro serviço de terceiros que possa coletar dados dos usuários.

O aplicativo não exibe anúncios e não possui integrações com redes de publicidade.

6 Segurança

Como nenhum dado é transmitido pela internet, o risco de interceptação de dados é inexistente. Seus arquivos e progresso de leitura são protegidos pelo sistema de segurança padrão do Android.

7 Crianças

O PageTurn não coleta dados de nenhum usuário, incluindo crianças menores de 13 anos. O aplicativo é seguro para uso por todas as idades, desde que o conteúdo importado seja apropriado para o usuário.

9 Alterações nesta política

Eventuais alterações nesta Política de Privacidade serão publicadas nesta página com a data de atualização revisada. Recomendamos revisitar esta página periodicamente.

O uso continuado do aplicativo após quaisquer alterações constitui aceitação da política atualizada.
